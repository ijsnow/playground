The Slow Reveal

This here is a web page, html and css just like the rest. But this web page is a bit different. Press the letter j. (Make sure your computer is "focused" on this page by clicking it.)

Whoa, something happened!

This page has some javascript that hides each line when the page loads.

A portion of this javascript asks the browser to wait for you to press the letter `j` to reveal the next line.

It uses functionality provided by the browser to find and interact with the elements on the page as well as listen for key presses.

There are several ways to interact with the html elements on a webpage. In this code, we are using capabilities provided by the browser. Another way would be to use jQuery. Can you rewrite the code so that it uses jQuery?

Writing a line of code is like writing a sentence. You can convey the same message with various combinations of words and no single combination is perfect. The code used to reveal the next line works but we do some extra work each time. Could you write it so that "less code" gets ran each time?
